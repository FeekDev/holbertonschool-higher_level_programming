The ``0-lookup`` module
======================

Using ``lookup``
-------------------

This is import ``lookup`` from the ``0-lookup.py`` module:

    >>> lookup = __import__('0-lookup').lookup

Example 1 in desorganized:

    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

example 2 asceding sort:

    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> my_list.print_sorted()
    >>> print(my_list)
    [1, 2, 3, 4, 5]
